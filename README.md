# 兔子洞

1. 收集个人记录：社交关系、文章、书籍、电影、音乐、标签1
2. 生成个人库及图谱


## 安装使用

1. 下载源码，导入 chrome，导入方法：https://notes.bluetech.top/website/install.jpeg
2. 本地设置一个文件夹，所有关系数据都导入到同一文件夹
3. 使用 obsidian 打开文件夹，查看图谱
